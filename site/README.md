# WisprDuck Site

This folder contains the marketing site built with Vite + React and Tailwind CSS.

## Development

```bash
cd site
npm install
npm run dev
```

## Build

```bash
cd site
npm run build
```

## Preview

```bash
cd site
npm run preview
```

## Deploy

```bash
cd site
npm run deploy
```

The deploy script runs a production build and then uses Wrangler to publish.
